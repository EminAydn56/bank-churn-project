Customer Churn Analysis Project (Python)
Python kullanarak bir banka müşteri terk (churn) veri seti üzerinde veri analizi gerçekleştirdim. Pandas ile veri temizleme ve keşifsel veri analizi (EDA) yaptım. Seaborn ve Matplotlib kullanarak görselleştirmeler oluşturup müşteri churn davranışlarını inceledim. Yaş, aktif üyelik durumu ve gelir gibi değişkenlerin müşteri kaybı üzerindeki etkilerini analiz ettim.

Customer Churn Analysis Project (Python)
Performed exploratory data analysis on a bank customer churn dataset using Python. Conducted data manipulation with Pandas and created visualizations using Seaborn and Matplotlib. Analyzed the impact of factors such as age, activity status, and estimated salary on customer churn behavior.
